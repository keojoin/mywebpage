ແມ່ນພາສາທີ່ໃຊ້ໃນການຂຽນເວັບເພຈ ໂດຍໃຊ້ Tag ໃນການກຳນົດການສະແດງຜົນ

HTML ແມ່ນຫຍັງ?

HTML5  HTML5 ຄື ພາສາມາກອັບ ທີ່ໃຊ້ສຳລັບຂຽນ website ຊຶ່ງ HTML5 ນີ້ເປັນພາສາທີ່ຖືກພັດທະນາຕໍ່ມາຈາກພາສາ HTML ແລະ ພັດທະນາຂຶ້ນມາໂດຍ WHATWG (The Web Hypertext Application Technology Working Group) ໂດຍໄດ້ມີການປັບເພີ່ມ Feature ຫຼາຍໆຢ່າງເຂົ້າມາເພື່ອໃຫ້ຜູ້ພັດທະນາສາມາດໃຊ້ງານໄດ້ງ່າຍຫລາຍຢິ່ງຂຶ້ນ

ຂໍ້ດີຂອງ HTML5

  1. ເວັບໄຊຕ໌ທີ່ສ້າງຈາກພາ¬ສາ HTML5 ສາມາດສະແດງຜົນໄດ້ກັບທຸກ web browser
  2. HTML5ຈະຊ່ວຍລຸດການໃຊ້ພວກປລັ໊ກອິນພິເສດຢ່າງພວກ Adobe Flash, Microsoft Silverlight, Apache Pivot ສະໜັບສະໜູນ ວິດີໂອ ແລະ ອົງຄ໌ປະກອບສຽງ ທັງຫມົດທັງ ສື່ມັນຕິມີເດຍຕ່າງໆຫລາຍຂຶ້ນ ໂດຍບໍ່ຕ້ອງໃຊ້ Flash
  3. ມີການຈັດການຂໍ້ຜິດພລາດທີ່ດີຂຶ້ນ
  4. ສະຄຣິບຕ໌ໃໝ່ ທີ່ຈະມາແທນທີ່ສະຄຣິບຕ໌ເດີມ (ຂຽນໂຄ້ດສັ້ນລົງ)
  5. HTML5 ມີຄວາມເປັນອິດສະຫລະສູງ (ຄ້າຍໆ XML )
  6. HTML5 ເຮັດວຽກຄອບຄູ່ກັບ CSS3 ໄດ້ດີ ຊ່ວຍໃຫ້ສາມາດເພີ່ມລູກຫລິ້ນຕ່າງໆເທິງເວັບໄຊຕ໌ໄດ້ສວຍງາມຫລາຍຢິ່ງຂຶ້ນ (CSS ຄືສ່ວນສະແດງຜົນ ທີ່ນັກອອກແບບສາມາດກຳນົດສີສັນ ຕຳແໜ່ງ ລັກສະນະເວລານຳເມົາສ໌ໄປແຫຍ່ແລ້ວມີກະຕ່າຍໂຜ່ອອກມາຈາກໂພງ ຫລື ຈັບກ້ອນວັດຖຸໃນໜ້າເວັບຯ ໃຫ້ຊິດຊ້າຍຊິດຂວາ ສ່ວນ CSS3 ຄືເວີຣ໌ຊັ່ນທີ່ 3 ຂອງ CSS )

features ໃໝ່ໆຂອງ HTML5

  1. Semantic Markup : ການເພີ່ມ Element ທີ່ ອ່ານງ່າຍຫລາຍຂຶ້ນ ແລະຊ່ວຍໃຫ້ ເຮົາເຮັດ SEO ໄດ້ມີປະສິດທິພາບຫລາຍຢິ່ງຂຶ້ນ
  2. Form Enhancements : ເພີ່ມຄວາມສາມາດຂອງ Form ຕ່າງໆ ບໍ່ວ່າຈະເປັນ Input type, Attribute ຫລື ແມ່ນແຕ່ Element
  3. Audio / Video: ຮອງຮັບການອ່ານໄຟລ໌ສຽງ ແລະ ວີດີໂອ ໂດຍບໍ່ຈຳເປັນຕ້ອງໃຊ້ Embed Code ຂອງ Third Party
  4. Canvas : ໃຊ້ໃນການວາດຮູບ ໂດຍຈຳເປັນຕ້ອງໃຊ້ Javascriptຊ່ວຍ
  5. ContentEditable : ສາມາດແກ້ໄຂ Content ໄດ້ໂດຍກົງຜ່ານທາງໜ້າເວັບ
